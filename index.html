API Example - PHP

In here you will find a simple PHP application that communicates with the Rare Carat API, sugin PHP. It is capable os listing, creating, updateing, and deleting resources. If you have any questions, feel free to email us at api@rarecarat.com.

Data Content Types

To maximize compatibility with existing systems, Rare Carat API accepts two formats for the data: CSV and JSON.

JSON

  • GET /diamonds (index)
  • POST /diamonds (create)
  • PATCH /diamonds/{id} (update single)
  • DELETE /diamonds/{id} (delete single)

CSV

  • GET /diamonds (index)
  • POST /diamonds (create)
  • PATCH /diamonds/{id} (update single)
  • DELETE /diamonds/{id} (delete single)
